-javascript folder contains the UI for the app.
-Create a virutal environment on the root folder
-Please read the comments in the main.py and set up the code as per your settings

Pre-requisites
-Install python version 3.8.7
-Run the fastAPI server on port 8000
-Download vs code

Commands to run the application
1)Create virtual environment
>>python -m venv env

2)activate it
>>.\env\Scripts\activate

3)Install dependencies
>> pip install -r requirements.txt 

4)Running the fastapi server
>>uvicorn main:app --reload

5)Run the html file inside Javascript-Speeach-to-Text on chrome
